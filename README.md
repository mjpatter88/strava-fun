# strava-fun
Have fun with strava data

## Run local
* Manually download dashbaord html as `Dashboard_Strava.html`
* `poetry run python scrape.py`

## Run Scrape (not yet implemented)
* `poetry run python scrape.py`

## Run tests
* `poetry run pytest`
* `poetry run ptw .`
